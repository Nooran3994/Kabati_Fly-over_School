## Typography

### Font Family
**Open Sans** - A clean, modern sans-serif font for professional appearance and excellent readability

- **Source**: Google Fonts (https://fonts.google.com/specimen/Open+Sans)
- **Weights Used**: 400 (Regular), 600 (Semibold), 700 (Bold)
- **Fallback**: `sans-serif`

### Font Sizes and Weights

#### Headings
- **H1** (Page Titles): 48px (3rem) on desktop, 36px (2.25rem) on mobile | Weight: 700 Bold
- **H2** (Section Headers): 36px (1.875rem) on desktop, 28px (1.75rem) on mobile | Weight: 700 Bold
- **H3** (Subsections): 24px (1.5rem) | Weight: 600 Semibold
- **H4** (Card Titles): 20px (1.25rem) | Weight: 600 Semibold

#### Body Text
- **Large Paragraph**: 18px (1.125rem) | Weight: 400 Regular | Line Height: 1.75rem
- **Body Paragraph**: 16px (1rem) | Weight: 400 Regular | Line Height: 1.625rem
- **Small Text**: 14px (0.875rem) | Weight: 400 Regular
- **Label/Caption**: 12px (0.75rem) | Weight: 600 Semibold

#### Links and CTAs
- **Standard Link**: 16px (1rem) | Weight: 600 Semibold | Color: Green (#22c55e)
- **Button Text**: 16px (1rem) | Weight: 600 Semibold | Color: White on Green background

### Line Spacing
- **Headings**: 120% (1.2 line height)
- **Body Text**: 150% (1.5-1.625 line height)
- **Large Text**: 175% (1.75 line height)

## Spacing System

The project uses an 8px-based spacing system for consistency and alignment.

### Spacing Scale
- **2xs**: 4px (0.25rem) - Very tight spacing
- **xs**: 8px (0.5rem) - Minimal spacing
- **sm**: 12px (0.75rem) - Small spacing
- **md**: 16px (1rem) - Standard spacing
- **lg**: 24px (1.5rem) - Large spacing
- **xl**: 32px (2rem) - Extra large spacing
- **2xl**: 48px (3rem) - Double extra large
- **3xl**: 64px (4rem) - Triple extra large

### Common Spacing Applications
- **Padding within cards**: 24px (lg)
- **Margin between sections**: 64px (3xl)
- **Margin between elements**: 16px-24px (md-lg)
- **Gap in grids**: 24px-32px (lg-xl)

## Accessibility Standards

### Color Contrast
- **Minimum WCAG AA**: 4.5:1 for text, 3:1 for graphics
- **WCAG AAA**: 7:1 for text
- All text colors meet accessibility standards

### Text Readability
- Minimum font size: 14px for body text
- Clear focus indicators for keyboard navigation
- Semantic HTML structure

### Forms
- All form inputs have associated labels
- Clear error messages
- Focus states clearly visible

---

## Interaction Patterns

### Hover States
- All interactive elements have clear hover states
- Smooth transitions (150-200ms)
- Color change and/or shadow enhancement

### Active States
- Navigation: Green text with bottom border
- Buttons: Darker background color

### Focus States
- Visible outline for keyboard navigation
- High contrast indicators

### Loading & Transitions
- Smooth fade-ins for content
- Clear loading indicators
- No jarring movements
